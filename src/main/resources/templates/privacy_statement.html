3. Erfassung von Daten

Wenn Sie sich in unserer Mobile App einen Account anlegen, so müssen Sie hierzu aktiv einen Benutzernamen, eine E-Mail-Adresse und ein Passwort angeben. Der Benutzername und die E-Mail-Adresse werden auf unserem Server abgespeichert. Des Weiteren werden von Ihnen gemachte „Gefällt mir“-Angaben und Kommentare ebenfalls auf dem Server gespeichert und ihrem Account zugewiesen.

4. Verwendung der Daten

Wenn Sie in unserer Mobile App eine Visual Novel kommentieren, so wird ihr Kommentar zusammen mit ihrem Benutzernamen angezeigt. Außerdem können Sie sich mit ihrem Benutzernamen einloggen.

Die E-Mail-Adresse wird verwendet, um unsere Richtlinie zu unterstützen, dass jede benutzende Person höchstens einen Account besitzt. Daher werden Sie gebet bei der Registrierung eine E-Mail-Adresse anzugeben und diese zu bestätigen. Außerdem wird die E-Mail-Adresse verwenden, um Ihnen Anweisungen zur Passwort-Wiederherstellung zu senden, sofern Sie mal ihr Passwort vergessen sollten. Außerdem können Sie sich mit ihrer E-Mail-Adresse einloggen. Für andere Nutzende ist Ihre E-Mail-Adresse nicht sichtbar. Ihr Benutzername und Ihre E-Mail-Adresse werden vertraulich behandelt und nicht an Dritte weitergegeben. Sie werden ausschließlich zu den oben genannten Zwecken verwendet.

5. Weitergabe von Daten

Ihre Daten werden ausschließlich zu den oben genannten Zwecken verwendet und nicht an Dritte weitergegeben.

6. Löschung von Daten

Sie können jederzeit über unsere Mobil App Ihren Account löschen. Wenn Sie dies tun, werden sofort alle mit ihrem Account verbundenen Daten gelöscht.
